React Callback Color Changer

A simple React + TypeScript project that demonstrates how to pass data from a child component to a parent component using callback functions.

Features

React functional components
TypeScript interfaces and props
Parent-to-child prop passing
Child-to-parent communication via callbacks
State management with useState
Dynamic background color updates

How It Works

The user enters a color in the child component (Callback.tsx).
The child component sends the color value to the parent using a callback function.
The parent component updates its state.
The page background changes to the selected color.

Technologies Used

React
TypeScript
Vite

Learning Objectives

Understanding React props
Using callback functions between components
Managing state with hooks
Building reusable TypeScript components
